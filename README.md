## Features

- **Emoji Search**: Search emojis by name, category, or tags in multiple languages
- **Emoji Categories**: Browse emojis organized by Unicode categories
- **Copy/Insert Modes**: Copy to clipboard or insert directly into text
- **SVG Import**: Import emoji SVGs as curve objects (View3D only)
- **Multi-Editor Support**: Works in View3D, Text Editor, and Geometry Nodes
- **Math Symbols**: Quick access to mathematical symbols and sub/superscripts

## Installation

1. Download the addon from...  (coming soon)
2. In Blender: **Edit > Preferences > Add-ons > Install**
3. Enable "Mesh Writer"

## Where to Find the Addon

- **View3D** > Sidebar (N) > Mesh Writer tab
- **Text Editor** > Sidebar (N) > Mesh Writer tab
- **Geometry Nodes** > Sidebar (N) > Mesh Writer tab
- **Shader Node Editor**  > Sidebar (N) > Mesh Writer tab
- **Compositor Node Editor** > Sidebar (N) > Mesh Writer tab
